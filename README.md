# Instruções de execução:
-  Instale a biblioteca zstandard: "sudo apt install python3-zstandard"
-  Certifique-se de conhecer o diretório do arquivo a ser lido, ou coloque-o no mesmo diretório do programa.
-  Para rodar um algoritmo em específico ou o programa main, siga a seguinte estrutura: "python3 <nome_do_programa.py> <nome/diretorio_do_arquivo> <tamanho_da_memoria> <tamanho_da_pagina>
